# Resilia Módulo 3: Projeto Individual "Estão servidos?"
Servidor JSON (Json-Server) contendo 3 rotas com a possibilidade de realizar GET, POST, PUT e DELETE.

## Tecnologias utilizadas:
- **Node**
- **Json-Server**
- **JavaScript**
